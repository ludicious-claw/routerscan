# Description

Router Scan can locate and identify various devices from a large number of known routers and, most importantly, extract useful information from them—specifically, wireless network specifications: the access point’s security method (encryption), access point name (SSID), and access point key (password). It also retrieves information about the WAN connection (useful when scanning a local network) and displays the router’s brand and model. Information is obtained in two possible ways: the program will attempt to find a username/password combination for the router from a list of standard passwords, thereby gaining access. Alternatively, it will exploit non-destructive vulnerabilities (or bugs) specific to a particular router model, allowing it to obtain the necessary information and/or bypass the authentication process.
